Installation
------------

```
$ git clone git@github.com:michaellennox/makers_idea_hub.git
$ cd makers_idea_hub
$ bundle
$ rake db:create
$ rake db:migrate
$ rails s
```
Visit: [http://localhost:3000](http://localhost:3000) on your machine.
